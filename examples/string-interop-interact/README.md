## Rust–C++ Interoperability Example

This example demonstrates basic interoperability between Rust and C++. It shows how Rust functionality can be exposed through a C-compatible interface and called from a C++ application.

## What This Example Shows

* Building Rust and C++ code in the same project
* Exposing Rust functions through an FFI boundary
* Calling Rust code from C++
* Linking Rust and C++ components together

## Running the Example

Build and run the example using the instructions provided in the repository root.

## Purpose

This example is intended as a minimal reference for developers who want to understand the fundamentals of Rust–C++ interoperability and mixed-language project integration.
